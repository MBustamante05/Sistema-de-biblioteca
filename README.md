# Sistema de Biblioteca

### Requerimientos
- [x] El sistema permitirá mostrar los libros disponibles
- [x] El sistema permitirá buscar un libro y mostrar su información
- [x] EL sistema permitirá marcar un libro como no disponible
- [x] El sistema permitirá al admin agregar un libro
- [x] El sistema permitirá al admin quitar un libro
- [x] El sistema permitirá prestar libros disponibles
- [x] El sistema permitirá devolver libros prestados
- [x] El sistema permitirá mostrar los libros prestados al admin y su información.
